БЛОК 2: ЕСЛИ СЕТЬ НЕ РАБОТАЕТ - НАСТРОЙКА IP
================================================================================

Сначала узнайте имя сетевого интерфейса:

ip link show

Вы увидите что-то вроде: eno1, eth0, enp0s1 и т.д.
Запомните имя интерфейса (например: eno1)

Затем выполните (ЗАМЕНИТЕ eno1 на ваш интерфейс):

sudo ip addr add YOUR_LAN_HOST/24 dev eno1
sudo ip link set eno1 up
sudo ip route add default via 192.168.1.1
echo "nameserver 8.8.8.8" | sudo tee /etc/resolv.conf

Проверка:
ping -c 3 192.168.1.1
ping -c 3 8.8.8.8
