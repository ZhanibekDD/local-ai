=================================================================
  СИСТЕМА ПАМЯТИ И КОНТЕКСТА ДИАЛОГА
=================================================================

✅ ЧТО ДОБАВЛЕНО:
-----------------
• Память диалога - бот помнит предыдущие сообщения
• Контекст разговора - можно продолжать тему
• Новый чат - кнопка для очистки истории
• Команда /clear - начать диалог заново


🧠 КАК РАБОТАЕТ ПАМЯТЬ:
-----------------------
1. Бот сохраняет последние 20 сообщений каждого пользователя
2. При новом запросе добавляет контекст предыдущих сообщений
3. Модель видит всю историю и может продолжать разговор
4. Каждый пользователь имеет свою отдельную историю


📝 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ:
-------------------------

Пример 1 - Продолжение темы:
  Вы: "Расскажи о Python"
  Бот: [рассказывает о Python]
  Вы: "А какие у него недостатки?"
  Бот: [понимает что речь о Python, отвечает о недостатках]

Пример 2 - Уточняющие вопросы:
  Вы: "Как создать REST API?"
  Бот: [объясняет]
  Вы: "А какой фреймворк лучше использовать?"
  Бот: [понимает контекст про REST API]
  Вы: "Покажи пример кода"
  Бот: [показывает код для REST API]

Пример 3 - Многошаговые задачи:
  Вы: "Помоги написать функцию сортировки"
  Бот: [пишет функцию]
  Вы: "Добавь обработку ошибок"
  Бот: [дополняет код обработкой ошибок]
  Вы: "Теперь добавь тесты"
  Бот: [добавляет тесты к функции]


🗑️ НОВЫЙ ЧАТ:
--------------
Когда использовать:
  • Хотите начать новую тему
  • Бот путается из-за старого контекста
  • Закончили одну задачу, начинаете другую
  • История стала слишком длинной

Как очистить:
  1. Кнопка "🗑️ Новый чат" в меню
  2. Команда /clear
  3. Команда /start (показывает кнопку)


⚙️ ТЕХНИЧЕСКИЕ ДЕТАЛИ:
-----------------------
• Хранится: последние 20 сообщений
• Формат: пары "вопрос-ответ"
• Хранилище: в памяти бота (не на диске)
• Приватность: каждый пользователь видит только свою историю


🎯 КОМАНДЫ:
-----------
/start  - главное меню (есть кнопка "Новый чат")
/clear  - очистить историю диалога
/ask    - задать вопрос (с контекстом)
/code   - генерация кода
/models - список моделей
/status - статус сервера
/help   - помощь


📊 ОГРАНИЧЕНИЯ:
---------------
• История хранится только пока бот работает
• При перезапуске бота история стирается
• Максимум 20 сообщений в истории
• Контекст не работает для изображений (пока)


🚀 ЗАПУСК ОБНОВЛЕННОГО БОТА:
-----------------------------
1. Остановите старого бота (Ctrl+C)
2. Запустите: python telegram-bot-local.py
3. Или: .\start-all.ps1


🧪 ТЕСТИРОВАНИЕ ПАМЯТИ:
------------------------

Тест 1 - Простая память:
  Вы: "Меня зовут Иван"
  Бот: [отвечает]
  Вы: "Как меня зовут?"
  Бот: [должен ответить "Иван"]

Тест 2 - Контекст темы:
  Вы: "Расскажи о машинном обучении"
  Бот: [рассказывает]
  Вы: "Приведи примеры"
  Бот: [приводит примеры ML]

Тест 3 - Многошаговая задача:
  Вы: "Напиши функцию факториала"
  Бот: [пишет код]
  Вы: "Оптимизируй её"
  Бот: [оптимизирует ту же функцию]

Тест 4 - Новый чат:
  Вы: [после тестов выше]
  Вы: /clear
  Бот: "История очищена"
  Вы: "Как меня зовут?"
  Бот: [не должен помнить "Иван"]
