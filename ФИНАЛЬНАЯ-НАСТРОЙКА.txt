ШАГ 2: Запуск Telegram бота
----------------------------
На вашем Windows компьютере:

Вариант А (автоматически):

Вариант Б (вручную):
  Терминал 1: ssh -p 42 -L 11434:localhost:11434 user@YOUR_PUBLIC_HOST
  Терминал 2: python telegram-bot-local.py

=================================================================
  ДОПОЛНИТЕЛЬНОЕ УСКОРЕНИЕ (опционально)
=================================================================

Если все еще медленно, можно:

1. Использовать меньшие модели:
   ollama pull qwen2.5:32b
   ollama pull deepseek-coder:6.7b

2. Использовать квантизацию Q4:
   ollama pull qwen2.5:72b-instruct-q4_K_M

3. Увеличить VRAM для кэша:
   OLLAMA_KV_CACHE_TYPE=f16
