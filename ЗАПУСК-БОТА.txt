  ИСПОЛЬЗОВАНИЕ В TELEGRAM
=================================================================

1. Откройте Telegram
2. Найдите вашего бота (используйте @BotFather для username)
3. Отправьте /start
4. Выберите действие кнопками или напишите сообщение

Примеры:
  💬 Привет! Расскажи о квантовых компьютерах
  💻 /code Напиши API на FastAPI

  📊 /status


=================================================================
  ЕСЛИ НЕ РАБОТАЕТ
=================================================================

1. Проверьте SSH туннель:
   curl http://localhost:11434/api/tags
   
2. Проверьте что Ollama работает на сервере:
   ssh -p 42 user@YOUR_PUBLIC_HOST "systemctl status ollama"
   
3. Проверьте логи бота в терминале

4. Перезапустите все:
   - Остановите бота (Ctrl+C)
   - Остановите SSH туннель (Ctrl+C)
   - Запустите start-all.ps1 снова
