=================================================================
  УСТАНОВКА QWEN-PRO - ЖЕСТКИЙ SYSTEM PROMPT
=================================================================

ЧТО ДЕЛАЕТ:
-----------
✓ Жесткий system prompt для точного следования инструкциям
✓ Few-shot примеры правильных ответов
✓ Обучение отвечать:
  - Строго по пунктам
  - Строго в JSON, если попросили
  - Без лишних объяснений
  - Без фантазий


УСТАНОВКА НА СЕРВЕРЕ:
----------------------
Скопируйте эти команды в SSH терминал:

cat > ~/create-qwen-pro.sh << 'SCRIPTEND'
#!/bin/bash
echo "Создание qwen-pro..."

cat > /tmp/Modelfile-qwen-pro << 'EOF'
FROM qwen2.5:72b

PARAMETER temperature 0.7
PARAMETER top_p 0.9
PARAMETER top_k 40
PARAMETER num_ctx 8192
PARAMETER num_gpu 1
PARAMETER repeat_penalty 1.1

SYSTEM """Ты профессиональный AI ассистент. Твоя задача - точно следовать инструкциям.

СТРОГИЕ ПРАВИЛА:
1. ВСЕГДА отвечай ТОЛЬКО на русском языке
2. Если просят JSON - отвечай ТОЛЬКО JSON, без текста
3. Если просят по пунктам - отвечай строго по пунктам
4. Если просят кратко - отвечай кратко
5. НЕ придумывай - если не знаешь, скажи "Не знаю"
6. НЕ добавляй лишнее
7. Следуй формату ответа

ПРИМЕРЫ:

JSON:
Пользователь: "Дай JSON о Москве"
Ты: {"город": "Москва", "страна": "Россия", "население": 12500000}

По пунктам:
Пользователь: "3 преимущества Python"
Ты: 
1. Простой синтаксис
2. Большая экосистема
3. Универсальность

Кратко:
Пользователь: "Кратко: что такое API?"
Ты: API - интерфейс для взаимодействия программ.

Честность:
Пользователь: "Погода в Москве?"
Ты: Не знаю текущую погоду, нет доступа к данным.

Точность важнее многословности!"""
EOF

ollama create qwen-pro -f /tmp/Modelfile-qwen-pro
echo "Готово!"
ollama list
SCRIPTEND

chmod +x ~/create-qwen-pro.sh
bash ~/create-qwen-pro.sh


ТЕСТИРОВАНИЕ:
-------------
После создания протестируйте:

# Тест 1 - JSON
ollama run qwen-pro "Дай информацию о Python в JSON: название, год, автор"

# Тест 2 - По пунктам
ollama run qwen-pro "Перечисли 5 преимуществ Linux"

# Тест 3 - Краткий ответ
ollama run qwen-pro "Кратко: что такое Docker?"

# Тест 4 - Честность
ollama run qwen-pro "Какая температура в Москве сейчас?"


ОЖИДАЕМЫЕ РЕЗУЛЬТАТЫ:
---------------------
✓ Тест 1: Чистый JSON без лишнего текста
✓ Тест 2: Строго 5 пунктов, без вступлений
✓ Тест 3: Одно предложение
✓ Тест 4: "Не знаю" вместо выдумки


ИСПОЛЬЗОВАНИЕ В БОТЕ:
----------------------
Бот автоматически будет использовать qwen-pro если модель доступна.

Приоритет моделей:
  1. qwen-pro (лучший)
  2. qwen-ru (хороший)
  3. qwen2.5:72b (базовый)


ТЕСТЫ ДЛЯ TELEGRAM БОТА:
-------------------------
После установки отправьте в бота:

1. "Дай список из 3 языков программирования в JSON"
   Ожидается: {"языки": ["Python", "JavaScript", "Java"]}

2. "Перечисли по пунктам 4 преимущества AI"
   Ожидается: Строго 4 пункта

3. "Кратко: что такое машинное обучение?"
   Ожидается: 1-2 предложения

4. "Сколько стоит биткоин сейчас?"
   Ожидается: "Не знаю точную цену..."


ЕСЛИ НЕ РАБОТАЕТ:
------------------
1. Проверьте что модель создана: ollama list
2. Перезапустите бота
3. Проверьте логи бота
