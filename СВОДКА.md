## Что нужно сделать

### ШАГ 1: Получить доступ к серверу

Выберите один из способов:

#### А) Через iDRAC9 (лучший вариант)
1. Откройте браузер
2. Перейдите: `https://YOUR_LAN_HOST` или `https://192.168.0.120`
3. Войдите: `root` / `<BMC password per vendor>` (или с наклейки сервера)
4. Откройте Virtual Console
5. Включите сервер если выключен
6. Войдите в Ubuntu: `youruser` / `***REDACTED***`

#### Б) Физический доступ
1. Подключите монитор и клавиатуру к серверу
2. Включите сервер
3. Войдите: `youruser` / `***REDACTED***`

#### В) Проверьте роутер
1. Зайдите в роутер: `http://192.168.1.1`
2. Посмотрите список подключенных устройств
3. Найдите Dell сервер и его реальный IP

### ШАГ 2: Настроить сеть на сервере

После получения доступа к консоли сервера:

```bash
# Проверка сетевых интерфейсов
ip addr show

# Проверка маршрутов
ip route show

# Проверка шлюза
ping 192.168.1.1

# Если сеть не настроена, настройте статический IP
sudo nano /etc/netplan/00-installer-config.yaml
```

Пример конфигурации:
```yaml
network:
  version: 2
  renderer: networkd
  ethernets:
    eno1:
      addresses:
        - YOUR_LAN_HOST/24
      routes:
        - to: default
          via: 192.168.1.1
      nameservers:
        addresses:
          - 8.8.8.8
          - 8.8.4.4
```

Применить:
```bash
sudo netplan apply
```

Проверить:
```bash
ip addr show
ping 192.168.1.1
ping 8.8.8.8
```

### ШАГ 3: Настроить SSH

```bash
# Проверка SSH
sudo systemctl status ssh

# Если не установлен
sudo apt update
sudo apt install -y openssh-server

# Запуск
sudo systemctl start ssh
sudo systemctl enable ssh

# Проверка
sudo netstat -tulpn | grep :22
```

### ШАГ 4: Запустить установку AI моделей

После настройки сети и SSH:

```bash

scp "C:\path\to\local-ai\install-commands.sh" user@YOUR_LAN_HOST:~/

# На сервере:
chmod +x install-commands.sh
./install-commands.sh
```

## После установки

### Использование моделей:

**1. Интерактивный режим (на сервере):**
```bash
ollama run qwen2.5:72b
>>> Привет! Напиши код на Python.
>>> /bye
```

**2. API из Windows (PowerShell):**
```powershell
$body = @{
    model = "qwen2.5:72b"
    prompt = "Напиши функцию сортировки"
    stream = $false
} | ConvertTo-Json

Invoke-RestMethod -Uri "http://YOUR_LAN_HOST:11434/api/generate" -Method Post -Body $body -ContentType "application/json"
```

**3. Python клиент:**
```bash
python ollama-client.py list
python ollama-client.py ask "qwen2.5:72b" "Привет!"
```
