КОМАНДЫ ДЛЯ ВЫПОЛНЕНИЯ НА СЕРВЕРЕ (через монитор и клавиатуру)
================================================================================

Попросите человека у сервера выполнить эти команды:

--------------------------------------------------------------------------------
1. ВХОД В СИСТЕМУ
--------------------------------------------------------------------------------

Username: youruser
Password: ***REDACTED***

--------------------------------------------------------------------------------
2. ПРОВЕРКА СЕТИ (скопировать весь блок)
--------------------------------------------------------------------------------

ip addr show
echo "---"
ip route show
echo "---"
ping -c 3 192.168.1.1
echo "---"
sudo systemctl status ssh

--------------------------------------------------------------------------------
3. ЕСЛИ СЕТЬ НЕ НАСТРОЕНА
--------------------------------------------------------------------------------

Узнайте имя интерфейса:
ip link show

Предположим интерфейс называется eno1 (замените если другой):

sudo ip addr add YOUR_LAN_HOST/24 dev eno1
sudo ip link set eno1 up
sudo ip route add default via 192.168.1.1

Проверка:
ping 192.168.1.1

--------------------------------------------------------------------------------
4. НАСТРОЙКА SSH
--------------------------------------------------------------------------------

sudo systemctl start ssh
sudo systemctl enable ssh
sudo ufw allow 22/tcp

--------------------------------------------------------------------------------
5. ПРОВЕРКА ИЗ WINDOWS
--------------------------------------------------------------------------------

После выполнения команд выше, попробуйте с Windows:

ping YOUR_LAN_HOST
ssh user@YOUR_LAN_HOST

--------------------------------------------------------------------------------
6. ЕСЛИ ПОДКЛЮЧЕНИЕ РАБОТАЕТ - УСТАНОВКА AI
--------------------------------------------------------------------------------

Выполните на сервере:

curl -fsSL https://ollama.com/install.sh | sh

sudo mkdir -p /etc/systemd/system/ollama.service.d
echo '[Service]' | sudo tee /etc/systemd/system/ollama.service.d/override.conf
echo 'Environment="OLLAMA_HOST=0.0.0.0:11434"' | sudo tee -a /etc/systemd/system/ollama.service.d/override.conf

sudo systemctl daemon-reload
sudo systemctl restart ollama
sudo systemctl enable ollama

sudo apt install -y nvidia-driver-535
sudo reboot

После перезагрузки:

ollama pull qwen2.5:72b
ollama pull llama3.2-vision:90b
