# Безопасность репозитория

- **Не коммитьте** пароли, API-ключи, внутренние IP, токены ботов, приватные SSH-ключи.
- Скрипты установки ожидают секреты в **переменных окружения** (`SSH_PASSWORD`, `DEPLOY_HOST`, `TELEGRAM_BOT_TOKEN`, `OLLAMA_BASE_URL` и т.д.).
- Если секреты когда-либо попали в git, считайте их скомпрометированными: **смените** пароли/токены и при необходимости очистите историю (`git filter-repo`, BFG Repo-Cleaner).
- После переписывания истории все участники должны сделать **fresh clone** или `git fetch --all` + `git reset --hard origin/main`; старые хэши коммитов перестанут совпадать — нужен **force push** (`git push --force-with-lease`).
