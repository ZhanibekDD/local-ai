# Шаблон: сервер с Ollama (без реальных секретов)

## 1. Доступ

- SSH: `ssh -p <PORT> <user>@<hostname_or_ip>`
- Пароли и ключи — только у вас локально, не в git.

## 2. GPU

```bash
nvidia-smi
```

## 3. Ollama

```bash
curl -s http://127.0.0.1:11434/api/tags
```

При работе с другой машиной часто используют туннель:

```bash
ssh -L 11434:127.0.0.1:11434 -p <PORT> <user>@<server>
```

Тогда клиент на ПК обращается к `http://127.0.0.1:11434`.

## 4. Модели

```bash
ollama pull <model_name>
```

## 5. Файрвол

Откройте только нужные порты; Ollama по умолчанию слушает `127.0.0.1` — для удалённого доступа настройте явно и осознанно (`OLLAMA_HOST`, reverse proxy, TLS).
