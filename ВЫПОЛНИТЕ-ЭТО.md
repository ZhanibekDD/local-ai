### Шаг 3: Установите Ollama

Скопируйте и выполните весь блок:

```bash
curl -fsSL https://ollama.com/install.sh | sh && \
sudo mkdir -p /etc/systemd/system/ollama.service.d && \
echo '[Service]' | sudo tee /etc/systemd/system/ollama.service.d/override.conf && \
echo 'Environment="OLLAMA_HOST=0.0.0.0:11434"' | sudo tee -a /etc/systemd/system/ollama.service.d/override.conf && \
echo 'Environment="OLLAMA_ORIGINS=*"' | sudo tee -a /etc/systemd/system/ollama.service.d/override.conf && \
sudo systemctl daemon-reload && \
sudo systemctl restart ollama && \
sudo systemctl enable ollama && \
systemctl status ollama
```

## 📊 Использование из Windows

### PowerShell:

```powershell
$body = @{
    model = "qwen2.5:72b"
    prompt = "Напиши функцию на Python для работы с файлами"
    stream = $false
} | ConvertTo-Json

$response = Invoke-RestMethod -Uri "http://YOUR_PUBLIC_HOST:11434/api/generate" -Method Post -Body $body -ContentType "application/json"
Write-Host $response.response
```

**⚠️ Для работы API из интернета нужно добавить проброс порта 11434 в роутере!**

### Python:

```python
import requests

response = requests.post('http://YOUR_PUBLIC_HOST:11434/api/generate', json={
    'model': 'qwen2.5:72b',
    'prompt': 'Напиши Hello World на Python',
    'stream': False
})

print(response.json()['response'])
```

---

## 🔧 Проброс порта 11434 в роутере

Для доступа к API из интернета, добавьте в роутере:

| Внешний порт | IP | Внутренний порт | Описание |
|--------------|-----|-----------------|----------|
| 11434 | YOUR_LAN_HOST | 11434 | Ollama API |

Тогда API будет доступен: `http://YOUR_PUBLIC_HOST:11434`
