✅ ЧТО РЕАЛИЗОВАНО:
-------------------
• Постоянное хранение диалогов в SQLite базе
• Множество чатов для каждого пользователя
• Переключение между чатами
• Автоматическое название чата
• История не теряется при перезапуске
• Каждый пользователь видит только свои чаты

📝 КАК ИСПОЛЬЗОВАТЬ:
--------------------

1. НАЧАТЬ ДИАЛОГ:
   • Просто напишите сообщение
   • Автоматически создается чат
   • Название = первое сообщение

2. ПРОДОЛЖИТЬ ДИАЛОГ:
   • Пишите следующие сообщения
   • Бот помнит весь разговор
   • Можно уточнять и продолжать

3. ПОСМОТРЕТЬ ЧАТЫ:
   • Кнопка "💾 Мои чаты"
   • Команда /chats
   • Видите список всех диалогов

4. ПЕРЕКЛЮЧИТЬ ЧАТ:
   • Выберите чат из списка
   • Бот покажет последние сообщения
   • Продолжайте с того места

5. НОВЫЙ ЧАТ:
   • Кнопка "➕ Новый чат"
   • Старые чаты сохраняются
   • Можно вернуться к ним позже


🔄 ПРИМЕРЫ РАБОТЫ:
------------------

Пример 1 - Длинный диалог:
  Чат "Изучение Python":
    Вы: "Расскажи о Python"
    Бот: [рассказывает]
    Вы: "А какие библиотеки популярны?"
    Бот: [перечисляет библиотеки]
    Вы: "Покажи пример с pandas"
    Бот: [показывает код]
    [... можно продолжать бесконечно]

Пример 2 - Несколько чатов:
  Чат 1: "Изучение Python"
  Чат 2: "Создание веб-сайта"
  Чат 3: "Анализ данных"
  
  Можно переключаться между ними!

Пример 3 - Возврат к старому чату:
  1. Вчера: разговор о Python
  2. Сегодня: /chats -> выбрать чат о Python
  3. Продолжить: "Покажи еще примеры"
  4. Бот помнит весь вчерашний разговор!


💾 ХРАНЕНИЕ ДАННЫХ:
-------------------
• База данных: bot_chats.db (SQLite)
• Таблицы:
  - chats: список чатов
  - messages: все сообщения
  - active_chats: текущий активный чат
• Размер: ~1KB на 100 сообщений
• Хранится: на вашем Windows компьютере


🎯 КОМАНДЫ:
-----------
/start  - главное меню
/chats  - список чатов
/ask    - задать вопрос
/code   - генерация кода
/models - список моделей
/status - статус сервера
/help   - помощь


⚙️ ТЕХНИЧЕСКИЕ ДЕТАЛИ:
-----------------------
• SQLite база данных
• Автоматическое создание чатов
• Автоматическое название (первое сообщение)
• Хранение до 10 последних чатов в списке
• Использование последних 20 сообщений для контекста
• Каждый пользователь изолирован


🚀 ЗАПУСК НОВОЙ ВЕРСИИ:
------------------------

1. Остановите старого бота (Ctrl+C)

2. Запустите новую версию:
   python telegram-bot-advanced.py

3. Или обновите start-all.ps1:
   Замените telegram-bot-local.py на telegram-bot-advanced.py

4. Откройте Telegram и протестируйте!


🧪 ТЕСТЫ:
---------

Тест 1 - Память в одном чате:
  1. "Меня зовут Иван"
  2. "Как меня зовут?"
  Ожидается: "Иван"

Тест 2 - Несколько чатов:
  1. Создайте чат 1: "Расскажи о Python"
  2. Создайте чат 2: "Расскажи о Java"
  3. /chats -> выберите чат 1
  4. "Покажи пример кода"
  Ожидается: Код на Python (не Java!)

Тест 3 - Долгий диалог:
  1. Задайте 10 вопросов подряд
  2. На 11-м вопросе упомяните что-то из 1-го
  Ожидается: Бот помнит начало разговора
