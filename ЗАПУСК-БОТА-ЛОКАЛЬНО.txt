=================================================================
  ЗАПУСК TELEGRAM БОТА ЛОКАЛЬНО (через SSH туннель)
=================================================================

ПРОБЛЕМА: Telegram API заблокирован на сервере провайдером

РЕШЕНИЕ: Запустить бота на вашем Windows компьютере, 
         подключиться к Ollama на сервере через SSH туннель


ШАГ 1: Откройте НОВЫЙ терминал PowerShell и создайте SSH туннель
------------------------------------------------------------------
ssh -p 42 -L 11434:localhost:11434 user@YOUR_PUBLIC_HOST

(Пароль: REDACTED)

ВАЖНО: Оставьте этот терминал открытым! Не закрывайте его!


ШАГ 2: Откройте ВТОРОЙ терминал PowerShell и запустите бота
------------------------------------------------------------
cd "C:\Users\Администратор\Desktop\DNEPR SERVER"

python telegram-bot-local.py


ШАГ 3: Откройте Telegram и найдите вашего бота
-----------------------------------------------
Отправьте /start для начала работы


=================================================================
  КАК ЭТО РАБОТАЕТ
=================================================================

1. SSH туннель перенаправляет порт 11434 с сервера на ваш компьютер
2. Бот работает на вашем Windows и имеет доступ к Telegram
3. Запросы к AI моделям идут через туннель на сервер
4. Ответы возвращаются обратно через туннель

Схема:
  Telegram <-> Ваш Windows (бот) <-> SSH туннель <-> Сервер (Ollama)


=================================================================
  ПРОВЕРКА ТУННЕЛЯ
=================================================================

После создания туннеля проверьте его в третьем терминале:

curl http://localhost:11434/api/tags

Должен вернуть список моделей с сервера


=================================================================
  АВТОЗАПУСК (опционально)
=================================================================

Для постоянной работы можно:
1. Держать компьютер включенным с запущенным ботом
2. Или использовать VPN на сервере для обхода блокировки Telegram
