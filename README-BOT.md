# 🤖 Telegram Bot для Dell R760 AI Server

Красивый Telegram бот с кнопками для работы с AI моделями на сервере Dell R760 + NVIDIA A40.

## ✨ Возможности

### 💬 Текстовые запросы
- Общение на русском и английском языках
- Модель: **Qwen2.5 72B** (47GB)
- Контекст: до 128K токенов

### 💻 Генерация кода
- Поддержка всех популярных языков
- Модель: **DeepSeek Coder 33B**
- Оптимизирован для программирования

### 🖼️ Анализ изображений
- Описание содержимого
- Распознавание текста
- Модель: **Llama Vision 90B** (54GB)

### 🎨 Интерфейс
- Красивые кнопки для быстрого доступа
- Интуитивное меню
- Статус сервера в реальном времени

## 🚀 Быстрый старт

### Вариант 1: Автоматический запуск

```powershell

```

### Вариант 2: Ручной запуск

**Терминал 1** (SSH туннель):
```powershell

```
Пароль: `***REDACTED***`

**Терминал 2** (Бот):
```powershell
python telegram-bot-local.py
```

## 📱 Использование в Telegram

1. Откройте Telegram
2. Найдите вашего бота
3. Отправьте `/start`
4. Выберите действие кнопками!

### Команды

- `/start` - главное меню с кнопками
- `/ask <вопрос>` - задать вопрос
- `/code <задача>` - генерация кода
- `/models` - список моделей
- `/status` - статус сервера
- `/help` - помощь

### Примеры

**Текстовые запросы:**
```
Привет! Расскажи о квантовых компьютерах
/ask Что такое машинное обучение?
```

**Генерация кода:**
```
/code Напиши функцию факториала на Python
/code Создай REST API на FastAPI
```

**Анализ изображений:**
```
[Отправьте фото]
Подпись: Что на этом изображении?
```

## ⚠️ Важно

1. **SSH туннель должен быть запущен** перед стартом бота
2. **Не закрывайте окно** с SSH туннелем
3. **Telegram API заблокирован** на сервере - поэтому бот работает локально
4. **Держите компьютер включенным** для работы бота

## 🐛 Решение проблем

### Бот не запускается
```powershell
# Проверьте SSH туннель
curl http://localhost:11434/api/tags
```

### SSH туннель не работает
```powershell
# Проверьте подключение к серверу
ssh -p 42 user@YOUR_PUBLIC_HOST
```

### Модели не отвечают
```bash
# На сервере проверьте Ollama
ssh -p 42 user@YOUR_PUBLIC_HOST "systemctl status ollama"
ssh -p 42 user@YOUR_PUBLIC_HOST "ollama list"
```
